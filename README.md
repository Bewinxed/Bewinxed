# Omar Al Matar

**Paramedic ‚Üí Healthcare Informaticist ‚Üí Full-Stack Engineer**

I started as an Emergency Medical Dispatcher in 2014, spent years triaging calls and coordinating ambulances, then taught myself to code because the software we had to use was terrible. Now I build the tools I wished existed.

Currently Auth Engineer at **Supabase** (WebAuthn, SCIM v2) and Software Engineer at **HMC Ambulance Services** in Qatar. Still work as an active dispatcher 25% of my time / I build better because I understand the problems firsthand.

I have a knack for hitting a barrier while coding and thinking "how would this look for others who face this?" Most of my open source comes from that instinct.
